Backup

Copy your entire Shelf Suite folder to a safe location (Desktop, Documents, an external drive — anywhere you want). That folder contains everything: your personal configs, custom icons, themes, and the engine. That's your full backup.

To restore it, just paste it back into your Rainmeter Skins folder and refresh the skins.

Updating to a new version

When a new version of ShelfSuite is released:

  • Back up your custom icons first — If you've added or replaced any icons in the @Resources/Icons folder with your own, copy them somewhere safe. The update will overwrite any icon that has the same filename as one included in the new version.
  • Install the new .rmskin — Download and double-click the new .rmskin file. The installer uses merge mode automatically, so it only updates the engine, configurator, themes, and default icons without touching your personal config.lua files.
  • Restore your custom icons — Copy your backed-up icons back into the @Resources/Icons folder.
  • Refresh your skins — Right-click each shelf on your desktop and select "Refresh skin".

Your personal shortcuts, tabs, and items are stored in config.lua files inside each Shelf folder. The installer only includes config.example.lua files, so your personal configurations are never overwritten.

The only thing that may reset is the theme on each shelf (since Shelf.ini gets updated). You can change it back in seconds from the theme dropdown above.

Help

Getting Started

ShelfSuite is a modular desktop organizer for Rainmeter. Each shelf is an independent widget that displays tabs with clickable items — folders, apps, URLs, or anything you want quick access to.

This configurator lets you visually manage your shelves without editing config files manually. Changes are saved directly to your Rainmeter skin files.

1. Connect your folder

Go to Setup and click Connect Folder. Navigate to your Rainmeter Skins folder and select the Shelf Suite folder. This is usually at:

C:\Users\YourName\Documents\Rainmeter\Skins\Shelf Suite

Once connected, the configurator reads your existing shelf configurations and any changes you make will be saved automatically.

2. Create or edit shelves

Click "+ New Shelf" in the sidebar to create a shelf. Give it a name and pick a theme. The shelf appears in the sidebar and you can click it to start editing.

Each shelf is independent — it gets its own folder with a Shelf.ini (the Rainmeter skin) and a config.lua (your items and tabs). You can have as many shelves as you want.

3. Organize with tabs

Each shelf can have up to 5 tabs (categories). Use "+ Add Tab" to create tabs. Hover over a tab to see the rename and delete buttons. You can't delete the last tab.

4. Add items

Click "+ Add Item" to add a shortcut. Each item needs:

  • Label — The name shown below the icon (e.g., "Documents")
  • Action — What happens when you click it. Can be a folder path (C:\Users\YourName\Documents), an app path (C:\Program Files\App\app.exe), or a URL (https://github.com)
  • Icon — Pick from 19 built-in icons

Each tab supports up to 18 items (3 rows of 6). Items can be edited or deleted using the buttons on each item card.

5. Choose a theme

Each shelf has a theme dropdown in its header. Pick from 4 themes:

  • DeepOcean — Dark blue tones
  • Forest — Green and earth tones
  • Terracotta — Warm orange and brown
  • Obsidian — Dark gray and charcoal

Themes affect the background, tab colors, and text colors of each shelf independently.

6. Apply changes in Rainmeter

After making changes, right-click the modified shelf on your desktop and select "Refresh skin". For new shelves, you'll need to load them manually in Rainmeter: right-click the Rainmeter tray icon, go to Skins > Shelf Suite > ShelfX > Shelf.ini, and click to load it.

Available Icons

19 icons from Google Material Icons (Apache 2.0 license), 64x64px white on transparent: calculator, calendar, chat, cloud, code, download, file, game, link, mail, music, news, office, photo, settings, storage, terminal, video, web.

You can add custom icons by placing 64x64 PNG files in the @Resources/Icons folder. They won't appear in the dropdown but can be used by editing config.lua directly.

Troubleshooting

  • Shelf shows no content — Make sure you have a config.lua (or config.example.lua) in the shelf folder, then refresh the skin.
  • New shelf doesn't appear in Rainmeter — You need to manually load it: Rainmeter tray > Skins > Shelf Suite > ShelfX > Shelf.ini.
  • Changes not showing — Right-click the shelf and select "Refresh skin".
  • Icons look wrong — Icons should be 64x64 PNG, white on transparent. Rainmeter scales them to 40px.
